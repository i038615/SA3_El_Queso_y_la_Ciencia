¿Cuánto calor se necesita para hacer queso?

¿Sabías que...?

Para hacer queso, la leche debe pasteurizarse, es decir, calentarse para eliminar bacterias y hacerla segura.

Mira este video para descubrir por qué se pasteuriza la leche con la que se producirá el queso.

Después del video, responde:

  • ¿Por qué es importante calentar la leche antes de hacer queso?
  • ¿A qué temperatura se realiza la pasteurización en la industria?
  • ¿Cómo se transfiere el calor en este proceso?

Ejercicio Resuelto

Cuando hacemos queso, es importante pasteurizar la leche para eliminar microorganismos no deseados. Este proceso implica calentar la leche hasta una temperatura determinada. Pero… ¿cuánta energía térmica necesitamos para calentarla?

Vamos a calcularlo aplicando la fórmula de transferencia de calor.

Datos del problema:

  • Cantidad de leche: 1 litro (1000 g)
  • Temperatura inicial: 10 °C
  • Temperatura final: 72 °C
  • Capacidad calorífica de la leche: 3.9 J/g·°C

Fórmula del calor transferido:

Q=m⋅c⋅ΔTQ 

Donde:

Q = cantidad de calor (en julios, J)
m= masa de la leche (en gramos, g)
c = capacidad calorífica de la leche (en J/g·°C)
ΔT = cambio de temperatura (Temperatura final - Temperatura inicial)

Sustituimos los valores en la ecuación:

Q=1000⋅3,9⋅(72−10)

Q = 1000 · 3,9 · 62

Q=241.800 J
